How to create a new Website

  1. Clone your repository
  2. create a file called index.html and add code in
  3. Validate the code
  4. create a file called main.css and add code in
  5. Validate the code
  6. Start the development server using python -m SimpleHTTPServer
  7. Type localhost:8000 into the browser to check
  8. push the files you created
  9. Start the production server- sftp username@cmis-hsteacher
  10. Change the directory of the sftp using cd public_html.
  11. then, put(upload) index.html and main.css separately
